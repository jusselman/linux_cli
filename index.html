Linux Terms

/

Root Directory

/boot

Where Linux & boot loaders are kept. Kernal called vmlinuz

etc

config files
/etc/passwd - defines users
/etc/fstab - table of devices mounted, defines drives
/etc/hosts - network host names & IP addresses
/etc/init.d - scripts that start system at bootup

/bin
/usr/sbin

sbin directories contain programs for system admin

/usr

Provides a variety of support for other applications
/usr/share/x11 - support files for XWindows
/usr/src - source code files, if you installed the kernel src cod pkg, you'll find the entire Linux Kernel src here
