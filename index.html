Linux Terms

ls

Provides a list of the file in the current working directory

ls /bin

Lists the files in the /bin directory

ls -l

Lists the files in the working directory in long format

ls -l /etc /bin

Lists the files in the /bin and /etc in long format

ls -la ..

Lists all files (including the hidden ones which begin with a period, normally are hidden) in the parent of the working directory, long format style.

ls -la ..

Lists all files (including the hidden ones which begin with a period, normally are hidden) in the parent of the working directory, long format style.

A break-down of Long Format

-rw------- 1 bshotts bshotts 576 Apr 17 1998 weather.txt

The above from left to right is defined as File Permissions, Owner, Group, Size (bytes), Modification Time, File Name

File name

The name of the directory or file

Modification Time

The last time the file was modified. If modded more than 6 months ago, date and year will be displayed. If more recent, the time and day will be printed

Size

The size in bytes

Group

The name of the group that has file permissions in addition to the file's owner.

Owner

The name of the user who owns the file

File Permissions

Displays the file's access permissions. The first char is the type of file. The "-" means it's an ordinary file. A "d" signifies a directory

Less

A program that displays the content of a text file

Page Up or b

Scroll back one page

Page Down or space

Scroll forward one page

G

Go to the end of the text file

1G

Go to the beginning of the text file

/characters

Search forward in the text file for an occurrence of the specified chars

n

Repeat the previous search

h

Display a complete list less commands and options

q

Quit

File

Whilst navigating the linux environment, it's crucial to determine what type of data a file contains before viewing. File displays what type of file it is.

/

Root Directory

/boot

Where Linux & boot loaders are kept. Kernal called vmlinuz

etc

config files
/etc/passwd - defines users
/etc/fstab - table of devices mounted, defines drives
/etc/hosts - network host names & IP addresses
/etc/init.d - scripts that start system at bootup

/bin
/usr/sbin

sbin directories contain programs for system admin

/usr

Provides a variety of support for other applications
/usr/share/x11 - support files for XWindows
/usr/src - source code files, if you installed the kernel src cod pkg, you'll find the entire Linux Kernel src here
